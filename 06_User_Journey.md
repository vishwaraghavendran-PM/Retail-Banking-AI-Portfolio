## Journey Summary
A relationship manager wants to prepare a personalized portfolio recommendation for a customer in seconds.

## Steps
1. RM logs into CRM demo interface.  
2. Selects a customer ID (synthetic data).  
3. System fetches portfolio from Cosmos DB.  
4. AI Analysis Agent processes financial behavior.  
5. Compliance Agent approves recommendations.  
6. UI displays:
   - Churn risk  
   - Recommended products  
   - Suggested CTA  
7. RM triggers execution (task or email).  
8. Results logged for auditability.
