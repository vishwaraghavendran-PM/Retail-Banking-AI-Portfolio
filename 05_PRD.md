# Product Requirements Document (PRD)
Retail Banking Agentic CRM

## 1. Purpose
Deliver an AI-driven CRM assistant that automates personalized portfolio insight generation and actionable recommendations.

## 2. Users
- Relationship Managers  
- Customer Experience Teams  
- Portfolio Analysts  

## 3. Features (MVP)
- Customer portfolio ingestion  
- AI-driven analysis and recommendations  
- Churn-risk scoring  
- Compliance filtering  
- CRM task creation  
- UI for demo  

## 4. Non-Functional Requirements
- Latency < 3 seconds  
- 99% JSON response validity  
- Secure PII handling  
- AAD authentication (optional extension)  

## 5. Future Enhancements
- Multi-customer batch analysis  
- RAG-based personalization  
- Integration with Dynamics 365  
- Personalized marketing templates
